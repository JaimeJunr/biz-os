# MEMORY — Índice da memória estruturada

> Arquivos lidos automaticamente por slash commands e skills. Cada um curto, denso, atualizável.

- [empresa.md](empresa.md) — quem é o cliente, o que faz, contexto do negócio
- [estrategia.md](estrategia.md) — foco atual, metas trimestrais, prioridades
- [preferencias.md](preferencias.md) — tom de voz, palavras-chave, gostos, restrições
- [user_role.md](user_role.md) — quem é o decisor/usuário do biz-os e seu papel

## Convenções

- Arquivos planos em `_memoria/` são **estado vivo do cliente** — mude quando o cliente mudar.
- Histórico/provas sociais ficam em `dados/historico/` (não aqui).
- Transcrições de reuniões ficam em `dados/transcricoes/` (não aqui).
- O CLAUDE.md raiz é o "cérebro persistente"; este índice é o atalho rápido.
