# Histórico / Provas sociais

Coloque aqui tudo que vira "memória de longo prazo" do cliente:

- `cases/` — estudos de caso completos (1 arquivo por cliente final atendido)
- `depoimentos.md` — coletânea de prints e quotes autorizados
- `numeros.md` — métricas-chave acumuladas (faturamento, leads, conversão)
- `reunioes/` — atas curtas extraídas das transcrições

Regra: nada aqui é inventado. Tudo vem de transcrição, print ou planilha real.
